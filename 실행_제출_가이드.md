## PART 0. 터미널 & 에디터가 뭔지 모르겠다면

### 터미널이 뭐야?
**터미널(=명령 프롬프트, 콘솔)**은 마우스 클릭 대신 **글자로 명령을 입력해서 컴퓨터를 조작하는 화면**이에요.

**여는 방법:**
- **Mac**: `Cmd + Space` → "터미널" 검색 → Enter
- **Windows**: 시작 메뉴에서 "cmd" 또는 "PowerShell" 검색 → Enter

**기본 명령어:**
```bash
cd 폴더이름       # 그 폴더 안으로 이동한다
ls                # (Mac) 지금 폴더 안에 뭐가 있는지 본다
dir               # (Windows) 위와 같음
```

### 에디터(VS Code)가 뭐야?
**VS Code**는 코드 파일을 눈으로 보고 수정하는 프로그램이에요.

**설치:** https://code.visualstudio.com → Download → 설치

**폴더 여는 법:**
1. VS Code 실행
2. **File → Open Folder** (Mac은 Open...)
3. `tollgate` 폴더 선택 → 열기

**VS Code 안에서 터미널 여는 법:**
- 메뉴 맨 위 **Terminal → New Terminal**
- 또는 단축키 `Control` + `` ` `` (백틱)
- VS Code 화면 안쪽에서 안 뜨면, 화면 최상단(모니터 맨 위) 메뉴바에서 **Terminal** 눌러야 해요 (Mac은 메뉴바가 창 안이 아니라 화면 꼭대기에 있음)

앞으로 나오는 모든 명령어는 **이 터미널 창**에 입력하면 돼요.

## PART 1. 앱 실행해보기

### 준비물
- **Node.js** 18+ → https://nodejs.org 에서 LTS 버전 설치
- **MetaMask** 브라우저 확장 → https://metamask.io
- **Anthropic API 키** → https://console.anthropic.com

### 1단계. 압축 풀고 VS Code로 열기
`tollgate.tar.gz` 압축 풀고, VS Code에서 **File → Open Folder** → `tollgate` 선택

### 2단계. 터미널에서 설치
```bash
npm install
```

### 3단계. 환경변수 파일 만들기
```bash
cp .env.local.example .env.local
```
(Windows는 `copy .env.local.example .env.local`)

왼쪽에서 `.env.local` 클릭해서 열고 채우기:
```env
ANTHROPIC_API_KEY=sk-ant-여기에_본인_키
NEXT_PUBLIC_AGENT_TREASURY=0x여기에_본인_지갑주소
```
저장은 `Cmd/Ctrl + S`

### 4단계. 실행
```bash
npm run dev
```

터미널에 `http://localhost:3000` 뜨면 그 글자 위에 `Cmd`(Mac) / `Ctrl`(Windows) 누른 채로 클릭하면 브라우저가 자동으로 열려요. 안 열리면 그 주소를 복사해서 브라우저 주소창에 붙여넣으면 돼요.

> 랜딩 페이지(파도 배경)가 먼저 뜨고, **Enter Tollgate** 버튼 누르면 실제 앱으로 들어갑니다.

### 5단계. 지갑 연결 + 테스트넷 USDC 받기
1. 앱에서 **Connect Wallet** 클릭 → MetaMask가 Arc Testnet을 자동으로 추가/전환
2. 새 네트워크 추가 승인 창 뜨면 **Approve** 클릭
3. https://faucet.circle.com 접속 → 본인 지갑 주소 입력 → 테스트 USDC 받기

### 6단계. 실제로 결제해보기
1. 질문 입력
2. **"Pay 0.01 USDC → Get AI Answer"** 클릭
3. MetaMask에서 서명
4. 서버가 **온체인에서 결제를 직접 검증**한 다음 (Arc RPC로 실제 확인) AI 답변을 줍니다
5. 답변 카드 아래 **"✓ Verified on-chain"** 배지와 **"Answered in n.nns"** 배지가 뜨는지 확인

이게 뜨면 결제 → 온체인 검증 → AI 응답까지 전체 흐름이 정상 작동하는 거예요.

### 탭 둘러보기
앱 상단에 탭이 4개 있어요:
- **Agent**: 질문하고 결제하는 메인 화면
- **Network**: 전체 사용자의 누적 결제 통계, 상위 결제자 순위, 최근 활동
- **History**: 본인 지갑 기준 지금까지 결제한 쿼리 목록
- **API**: 선충전 크레딧 + API 키 발급. 매번 지갑 서명 안 하고 코드에서 바로 호출할 수 있게
- **Docs**: 이 결제 검증 패턴을 다른 앱에 붙이는 방법 코드 예시

### 새로 추가된 기능들 (v0.3)
- **지갑 메뉴**: 연결된 주소 클릭하면 주소 복사 / 익스플로러 보기 / 연결 해제 메뉴가 떠요
- **잘못된 네트워크 경고**: 지갑이 다른 체인에 있으면 노란 배너가 뜨고, 버튼 한 번으로 Arc로 전환돼요
- **실시간 네트워크 상태**: Stats 카드의 Network 칸에 Arc 최신 블록 번호가 실시간으로 떠요 (30초마다 갱신)
- **지갑 계정 전환 자동 감지**: MetaMask에서 계정 바꾸면 앱도 자동으로 따라가요

### API / 선충전 크레딧 (v0.4)
- **API 탭**에서 지갑으로 소액 결제하면 잔액이 충전돼요 (실제 온체인 결제 1번으로 크레딧 생성)
- 그 잔액으로 **API 키**가 발급되고, 이 키로 `/api/v1/query`를 호출하면 지갑 서명 없이 질문할 수 있어요
- curl 예시가 API 탭 안에 바로 있어요, 복사해서 터미널에 붙여넣으면 실제로 작동해요

## PART 3. Vercel 배포 (데모 URL 만들기)

1. https://vercel.com → GitHub 계정으로 로그인
2. **Add New → Project** → `tollgate` 저장소 선택 → **Import**
3. **Environment Variables**에 두 개 추가:

   | Name | Value |
   |---|---|
   | `ANTHROPIC_API_KEY` | sk-ant-... |
   | `NEXT_PUBLIC_AGENT_TREASURY` | 0x본인지갑 |

4. **Deploy** 클릭 → 1~2분 후 `https://tollgate-xxx.vercel.app` 생성됨

### 배포 URL 나온 후 꼭 할 것: 환경변수 하나 더 추가
1. Vercel 프로젝트 → **Settings → Environment Variables**
2. `NEXT_PUBLIC_SITE_URL` 추가하고 값은 방금 생성된 배포 URL로 (예: `https://tollgate-xxx.vercel.app`)
3. 저장 후 **Deployments** 탭에서 **Redeploy** 한 번 눌러주기

이거 안 하면 카톡/디스코드/트위터에 링크 공유했을 때 미리보기 카드 이미지가 안 뜰 수 있어요.

### 파비콘 & 공유 카드 확인하는 법
- **파비콘**: 배포된 사이트를 새 탭에서 열면 브라우저 탭에 톨게이트 아이콘이 떠요
- **공유 카드(OG 이미지)**: 배포 URL을 카카오톡이나 디스코드에 붙여넣어보면 검은 배경에 "AI answers, paid per query in USDC." 카드 이미지가 뜨는지 확인
- 안 뜨면 https://www.opengraph.xyz 같은 사이트에 배포 URL 넣어서 바로 확인 가능

### 커스텀 도메인 (선택사항)
`tollgate.xyz` 같은 도메인 사고 싶으면:
1. `namecheap.com` 또는 `porkbun.com`에서 검색 → 등록 가능하면 구매 ($1~15)
2. Vercel 프로젝트 → **Settings → Domains** → 도메인 입력 → Add
3. 안내되는 DNS 값을 도메인 구매처 DNS 설정에 입력 → 최대 1시간 내 반영

도메인 없이 Vercel 기본 URL로 제출해도 전혀 문제없어요.

### ⚠️ Associated smart contracts 칸 (중요)
Circle 팀 공지에 따라 솔직하게 이렇게 적으세요:

> Tollgate does not deploy its own protocol contracts. Payments are native USDC transfers on Arc testnet (USDC is Arc's native gas token), verified server-side via direct calls to Arc's JSON-RPC endpoint (eth_getTransactionByHash), checking recipient, amount, and confirmation status before serving paid content. No proprietary contracts were deployed for this submission. This is an early-stage MVP; we are actively iterating toward a lightweight settlement contract and Circle Wallets integration.

USDC 컨트랙트 주소를 "내 컨트랙트"라고 절대 적지 마세요.

## 자주 막히는 곳

- **npm install 에러**: Node.js 버전 확인 (`node -v` → 18 이상)
- **"Unrecognized chain ID" / "RPC does not match chainID"**: `src/lib/constants.ts`에서 chainId가 `"0x4CEF52"`로 되어 있는지 확인 (5042002의 정확한 hex 값)
- **MetaMask 연결 안 됨**: 브라우저 확장 설치 확인, 잠금 해제 확인
- **지갑 연결 시 확장 프로그램이 전체화면으로 커짐**: 정상 동작이에요. 새 네트워크 추가 승인 화면이라 그런 거고, **Approve** 누르면 다시 작아져요.
- **"Payment verification failed"**: 결제가 실제로 Arc Testnet에 확정됐는지 확인. 확정 직후라 아직 블록에 안 들어갔을 수 있음 (몇 초 후 재시도)
- **AI 답변 안 옴**: `.env.local`의 `ANTHROPIC_API_KEY` 확인. Vercel이면 환경변수 저장 후 재배포 필요
- **결제 실패**: 테스트넷 USDC 잔액 있는지 확인 (`faucet.circle.com`에서 받기)
